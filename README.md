# 🎬 Movie Ratings Analysis

An exploration of audience movie ratings to uncover trends in preferences and rating distributions.

## 📌 Overview
This project analyzes a movie ratings dataset using Python to understand how ratings are distributed and what patterns emerge in audience preferences.

## 🛠️ Tools & Libraries

- pandas
- NumPy
- Matplotlib / Seaborn (for visualization)

- Visualized rating patterns

- Learned to summarize large datasets into clear visual insights
